Vogo.Family Chat Widget - Direct Embed Demo

This page demonstrates how to embed the Vogo.Family Chat Widget directly on your website.

Note: The widget should appear in the bottom right corner of this page.

How to Embed the Widget

To embed the widget on your website, add the following code just before the closing </body> tag:

<!-- Vogo.Family Chat Widget -->
<div id="vogo-chat-root"></div>
<link href="https://vogo-chatbot.vogo.family/_next/static/css/e60b2bb6c1978960.css" rel="stylesheet">
<script src="https://vogo-chatbot.vogo.family/_next/static/chunks/webpack-4cdc0867cc5dd3b0.js"></script>
<script src="https://vogo-chatbot.vogo.family/_next/static/chunks/4bd1b696-640aabc2b96c0448.js"></script>
<script src="https://vogo-chatbot.vogo.family/_next/static/chunks/684-f668407047b25d15.js"></script>
<script src="https://vogo-chatbot.vogo.family/_next/static/chunks/main-app-116ab2d175890b3e.js"></script>
<script src="https://vogo-chatbot.vogo.family/_next/static/chunks/app/page-231765ea69699e49.js"></script>
<script src="https://vogo-chatbot.vogo.family/initialize-widget.js"></script>

Important Notes

  • The widget requires all the scripts to be loaded in the correct order.
  • The initialize-widget.js script is responsible for properly rendering the widget.
  • Make sure the vogo-chat-root div exists in your HTML before the scripts run.
