Vogo.Family Chat Widget Integration

This page demonstrates the Vogo.Family Chat Widget. The widget should appear in the bottom right corner of the page.

How to Embed the Widget

To embed the Vogo.Family Chat Widget on your WooCommerce site, add the following code just before the closing </body> tag:

<!-- Vogo.Family Chat Widget -->
<div id="vogo-chat-root"></div>
<script src="https://vogo-chatbot.vogo.family/public/embed.js"></script>

That's it! The widget will automatically load the necessary CSS and JavaScript files.
